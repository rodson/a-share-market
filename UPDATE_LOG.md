### 🎯 使用方法

1. **选择日期**
   ```
   在日期选择器中选择任意日期（2005-01-01 至 2025-10-31）
   ```

2. **查询数据**
   ```
   点击"查询数据"按钮
   ```

3. **查看结果**
   ```
   - 顶部卡片显示选中日期的估值指标
   - 图表中可以看到该日期在历史中的位置
   - 鼠标悬停查看精确数值
   ```

4. **分析判断**
   ```
   - 股债利差分位 < 30%：市场可能过热
   - 股债利差分位 > 70%：市场可能低估
   - PE分位 > 80%：估值偏高，需谨慎
   - PE分位 < 20%：估值偏低，可关注
   ```
